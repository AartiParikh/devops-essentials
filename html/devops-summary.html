DevOps: Summary

Summary

In this lesson you learned -

  • the basic components and tools of a deployment pipeline
  • how to describe DevOps - in terms of applying organizational, culture, tooling, and process changes to increase feedback loops
  • the patterns of effective DevOps culture through Westrum's organizational models and behaviors
  • how teams that embrace DevOps organize to reduce silos and increase feedback
  • one of the primary methods for embracing DevOps: Continuous Delivery, and how to you can implement an assemnbly line for regualrly delivering software to users to increase feedback

Delete AWS Resources
