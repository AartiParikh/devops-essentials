Deploying to EC2: Summary

Summary

In this lesson on creating and deploying AWS resources to EC2 you learned -

  • 3.1 Describe basic EC2 features
  • 3.2 Describe how to deploy applications using CodeDeploy
  • 3.3 Describe the architecture and launch the solution
  • 3.4 Provision AWS resources in CloudFormation
  • 3.5 Orchestrate solution in CodePipeline

Delete AWS Resources

To delete the AWS resources used in this lesson -

  • Go to the CloudFormation console and select the checkbox next to the stack you created in this exercise, select Actions, then Delete Stack.
  • Remove any S3 buckets and objects created during the exercises.
