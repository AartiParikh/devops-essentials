Static Website: Summary

Summary

In this lesson on creating and deploying AWS resources for a static website you learned -

  • 2.1 Configure course prerequisites
  • 2.2 Manually provision static website
  • 2.3 Describe how to orchestrate software workflows in CodePipeline
  • 2.4 Store software assets in CodeCommit
  • 2.5 Build and test software in CodeBuild
  • 2.6 Describe the architecture and launch the solution
  • 2.7 Provision AWS resources in CloudFormation
  • 2.8 Orchestrate solution in CodePipeline

Delete AWS Resources

To delete the AWS resources used in this lesson -

  • Go to the CloudFormation console and select the checkbox next to the stack you created in this exercise, select Actions, then Delete Stack.
  • Remove any S3 buckets and objects created during the exercises.
